The h1 tag is used to create the heading. There is an order of the headings according to their size.

The style tag is used to modify the things we use to create a website....currently, its used to place the heading in the centre by the code of "text-align" given above.

The h2 tag also creates the heading but of small size as compared to heading 1. Similar goes for the h3 tag, h4 tag, and so on.

The body tag is the main body which the main content is written.

The p tag refers to "paragraph".


The br tag is used to break the sentence and bring it to the next line. Just close the br tag after the sentence you want to break.

The sentence after the tag will be displayed on the next line.
